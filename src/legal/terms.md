# 메이플북 서비스 이용약관

## 제1조 (목적)
본 약관은 **메이플북**(이하 "서비스")이 제공하는 NEXON Open API 기반 캐릭터 데이터 조회 서비스의 이용과 관련하여, 서비스와 이용자 간의 권리, 의무, 책임 소재 및 기타 필요한 사항을 규정함을 목적으로 합니다.

## 제2조 (용어의 정의)
1. **"서비스"** 란 이용자가 NEXON Open API Key를 입력하여 넥슨 게임의 캐릭터 정보 및 관련 데이터를 조회할 수 있도록 제공되는 웹사이트(메이플북)를 의미합니다.
2. **"이용자"** 란 본 약관에 동의하고 서비스가 제공하는 기능을 이용하는 자를 의미합니다.
3. **"NEXON Open API Key"** 란 넥슨(NEXON)이 제공하는 데이터를 호출하기 위해 이용자가 넥슨 개발자 센터에서 직접 발급받은 고유 인증키를 의미합니다.

## 제3조 (약관의 효력 및 변경)
1. 본 약관은 이용자가 서비스를 최초로 이용함과 동시에 효력이 발생합니다.
2. 서비스는 관련 법령을 위배하지 않는 범위 내에서 본 약관을 개정할 수 있으며, 변경된 약관은 적용일자 7일 전(중요한 변경의 경우 30일 전)에 서비스 내 공지사항을 통해 안내합니다.

## 제4조 (NEXON Open API 및 지식재산권)
1. 본 서비스는 **㈜넥슨코리아**의 Open API를 활용하여 제작된 비영리 서비스입니다.
2. 서비스를 통해 제공되는 게임 내 캐릭터, 이미지, 세계관 등 모든 데이터의 지식재산권은 **㈜넥슨코리아** 에 귀속됩니다.
3. 본 서비스는 넥슨의 공식 서비스가 아니며, 넥슨의 '게임 IP 사용 가이드' 및 'NEXON Open API 이용약관'을 준수합니다.

## 제5조 (이용자의 의무 및 API Key 관리 책임)
1. 이용자는 본인의 **NEXON Open API Key** 를 자신의 책임하에 직접 관리해야 하며, 이를 타인에게 양도, 대여, 공유할 수 없습니다.
2. 이용자가 입력한 API Key는 NEXON Open API 이용약관에 따라 서버 데이터베이스에 저장되지 않으며, 이용자의 브라우저 쿠키에 암호화된 형태로만 보관됩니다. 이용자의 기기 보안 취약점, 관리 소홀이나 부주의로 인해 API Key가 유출되어 발생하는 피해와 책임은 전적으로 **이용자 본인**에게 있습니다.
3. 이용자는 서비스를 영리적 목적(데이터 재판매 등)으로 이용할 수 없으며, 불법적인 방법(해킹, 매크로 등)으로 서버에 부하를 주는 행위를 해서는 안 됩니다.

## 제6조 (서비스의 제공 및 중단)
1. 서비스는 연중무휴 1일 24시간 제공함을 원칙으로 합니다.
2. 단, 다음 각 호에 해당하는 경우 서비스 제공이 일시적으로 중단될 수 있으며, 이로 인한 불편에 대해 서비스는 책임을 지지 않습니다.
   - 넥슨(NEXON) Open API 서버의 점검, 장애 또는 정책 변경으로 인해 데이터 호출이 불가능한 경우
   - 메이플북 호스팅 서버(Cloudflare 등)의 점검, 교체 또는 통신 장애가 발생한 경우
   - 천재지변, 국가비상사태 등 불가항력적 사유가 발생한 경우

## 제7조 (비영리 운영 및 후원)
1. 본 서비스는 무료로 제공되는 비영리 프로젝트입니다.
2. 서비스 운영 및 서버 유지보수를 위하여 자발적인 **후원(Ko-fi 등)** 을 받을 수 있습니다.
3. 후원은 전적으로 이용자의 자발적 의사에 따르며, 후원 여부가 데이터 조회 결과나 핵심 서비스 이용 권한에 어떠한 차별적 영향도 미치지 않습니다.
4. 이용자가 미성년자(만 19세 미만)인 경우, 후원 진행 시 법정대리인(부모 등)의 동의를 얻은 것으로 간주합니다. 법정대리인의 동의 없이 이루어진 후원에 대해서는 관련 법령에 따라 취소 및 환불이 제한될 수 있습니다.
5. 후원금은 원칙적으로 서버 유지보수 및 서비스 운영을 위해 즉각 사용되므로, 단순 변심으로 인한 결제 취소 및 환불은 불가능합니다.

## 제8조 (면책 조항)
1. 서비스는 넥슨 Open API를 통해 데이터를 '있는 그대로' 중계하여 보여줄 뿐이며, 제공되는 데이터의 **정확성, 신뢰성, 완전성** 에 대해 어떠한 보증도 하지 않습니다.
2. 서비스는 무료로 제공되는 바, 서비스 이용과 관련하여 이용자에게 발생한 어떠한 손해(데이터 유실, 정신적 피해 등)에 대해서도 서비스 운영자는 고의 또는 중과실이 없는 한 법적 책임을 지지 않습니다.
3. 이용자가 제5조(이용자의 의무)를 위반하여 넥슨 또는 제3자에게 손해를 입힌 경우, 이용자는 자신의 비용과 책임으로 이를 배상해야 하며 운영자를 면책시켜야 합니다.

## 제9조 (이용 계약의 해지)
1. 이용자는 언제든지 마이페이지 등의 기능을 통해 탈퇴를 요청할 수 있으며, 서비스는 관련 법령 및 개인정보처리방침에 따라 이용자의 데이터를 지체 없이 파기합니다.
2. 이용자가 본 약관을 위반하거나 넥슨의 IP 사용 가이드를 위반한 사실이 적발될 경우, 서비스는 사전 통보 없이 해당 이용자의 계정을 정지하거나 삭제할 수 있습니다.

## 제10조 (준거법 및 관할법원)
본 약관의 해석 및 서비스 이용과 관련하여 발생한 분쟁에 대해서는 대한민국 법률을 적용하며, 소송이 제기될 경우 운영자의 주소지를 관할하는 법원을 전속 관할법원으로 합니다.

**부칙**
본 약관은 **2026년 3월 19일**부터 적용됩니다.
